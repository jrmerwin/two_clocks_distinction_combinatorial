# Methods appendix draft

## Pipeline summary

1. A fixed-grid DEU hook-validation sequence established a full-objective working lock and retained a tri-anchor ledger.
2. Stage 3B sampler-plumbing checks validated fixed reference and sampler reference consistency before MCMC.
3. Production DEU chains and native/baseline chains were run and readiness-gated.
4. A first posterior summary, robustness checks, matched native comparison, and internal evidence-estimator lineage were recorded.
5. This stage assembles an author-led preprint package without launching MCMC or computing new evidence.

## Claim boundary

The preprint may report the caveated posterior and internal estimator lineage, but it may not state peer review, model superiority, TT_lite inclusion in sampled likelihoods, or final publication-grade validation.
